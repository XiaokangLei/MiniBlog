# 微信小程序个人博客

## 简介

微信小程序搜索【九月Online】查看小程序具体内容。

MiniBlog是一款基于云开发的博客小程序，该小程序完全不依赖任何后端服务，无需自己的网站、服务器、域名等资源，只需要自行注册小程序账号即可。

源码全部开源。
